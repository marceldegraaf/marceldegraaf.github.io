My OS/X toolbar - the tools I daily use

In this post I’d like to walk you through my OS/X toolbar, to share the tools I daily use. Hopefully you’ll find these as useful as I do!

Without further ado:

Twitter – keeping me informed of what happens on Twitter.

Trailer – shows OS/X notifications for updates to Github Pull Request I’m involved in.

Chrome Notifications – desktop notifications from the Chrome browser.

F.lux – changes the color of my display according to the time of day. If you stare at a screen for most of your day, do yourself a favor and please install this. It’s awesome.

Elastics – quick statistics from (and access to) running Amazon Web Services EC2 instances.

Fantastical – a beautifully designed Mac calendar application.

Monosnap – everything Skitch used to provide, and much more. Really useful application to quickly snap screenshots (and annotate them when needed).

BreakTime – helps me take regular breaks from working.

BitTorrent Sync – file sharing application based on BitTorrent.

Isolator – I like only having the active window being visible. Isolator hides all non-active windows.

Alfred – launcher and productivity application. Really, use this.

Arq – locally encrypted backups to Amazon Web Services Glacier, S3, and other S3 compatible storage platforms.

Sparrow – the (unfortunately, abandoned) email client. Looking for a better alternative. Please let me know (@marceldegraaf) if you know a good email client I should try.

Screenhero – remote pair programming made easy. This application is great for working together remotely.

Dropbox – Duh.

1Password – manages all my passwords, credentials, and other secure bits of information. Seriously, use this.
